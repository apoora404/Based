XP 효율 · 수수료 계산기

이전/현재 누적 거래량·포인트로 사용 구간을 계산하고, Tier별 수수료를 반영해 총 수수료·리베이트(60%)·XP 효율·XP당 단가를 구합니다. (고급 옵션: 이전/현재 총 수수료(누적)로 이번 구간 총 수수료를 자동 계산)

▾ 고급 옵션(선택): 이전/현재 총 수수료(누적) 입력 → 사용 구간 총 수수료 자동 계산

※ 입력 시 두 값의 차이로 이번 구간의 총 수수료(리베이트 전)를 자동 계산합니다. 리베이트(60%)는 결과에서만 표시됩니다.

Perps · Tier 0 Maker/Taker: -

평균 수수료율은 기본적으로 Maker만 가정합니다. 총 수수료(누적 차이)를 입력하면 평균 수수료율과 추정 Taker 비중을 역산해 정확도를 높입니다. 리베이트 60%는 결과에서만 적용됩니다.
